# VocalGuard

Voice-based early screening prototype for Parkinson's-related speech deviations.

## Features
- Baseline speech profiling 
- Dataset-based comparision
- Real-time voice recording
- Acoustic feature extraction
- Deviation-based risk indicators

## Disclaimer
This tool is for reasearch and early screening support only.
It does not provide medical diagnosis.

## How to Run
pip install -r requirements.txt
streamlit run app.py
